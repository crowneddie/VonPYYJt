# 项目简介

## 核心功能
- 用户预约管理
- 管理员后台管理
- 预约信息展示
- 博物馆信息展示
- 预约规则设置

## 适用场景
- 博物馆预约管理
- 文化活动报名
- 公共资源预约

## 技术特点
- 使用Spring Boot作为后端框架，确保系统的高效运行和良好的扩展性。
- 采用Vue进行前端开发，提供流畅的用户体验。
- 系统基于小程序架构，便于用户随时随地预约。
